## Hello, world!

**alert 창 띄우기**

```html
<html>
    <body>
        <script>
        	alert("JavaScript!");
        </script>
    </body>
</html>
```

**외부 스크립트를 이용해 alert**

```html
<body>
    <script src="alert.js"></script>
</body>
```

`alert.js`

```js
alert("Javscript!");
```

## 변수와 상수

변수 가지고 놀기

```js
let admin, name;
name = "john";
admin = name;
console.log(admin);
```

올바른 이름 선택하기

```javascript
let planet = "earth";

let connectedUser = "son";
```

대문자 상수 올바로 사용하기

```javascript
const BIRTHDAY = '18.04.1982';

let age = someCode(BIRTHDAY);
```
