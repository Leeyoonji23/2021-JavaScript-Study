#### 첫 글자를 대문자로 변경하기
```javascript 
function ucFirst(str) {
  return str[0].toUpperCase() + str.slice(1);
}
alert( ucFirst("john") );
```

#### 문자열 줄이기
```javascript
function truncate(str ,maxlength) {
  return (str.length > maxlength) ?
  str.slice(0, maxlength - 1)+ '...' : str;
}
```
